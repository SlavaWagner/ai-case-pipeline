# ai-case-pipeline 🚀
> Autonomous AI Case-Schmiede & Content Forge for Google Ads Performance Analysis

`ai-case-pipeline` ist ein autonomer KI-Agent für die Antigravity CLI. Er durchsucht Google Ads Accounts nach historischen Durchbruch-Zeiträumen, führt tiefgreifende Feedback-Schleifen auf Keyword-, Asset-, Conversion-Wert- und Ziel-ROAS-Ebene durch und schmiedet fertige Marketing- und Content-Pakete auf Ihrem Desktop.

---

## 🌟 Hauptfunktionen

1. **Google Ads API Setup-Prozess**: Integriertes Setup für Customer ID, Developer Token, Client ID, Client Secret & Refresh Token.
2. **Antigravity CLI Architecture**: Sämtliche KI-Verarbeitungen laufen direkt über die Antigravity CLI Infrastruktur.
3. **CLI Terminal Dashboard**: Interaktive Steuerung direkt im Terminal (kein Browser nötig).
4. **Gewinn-Zeitraum-Scanner**: Automatische Identifikation von Performance-Spikes.
5. **Deep-Dive Feedback Engine**: Analyse von Keyword-Strukturen, Negativ-Loops, Asset-Headlines und Conversion-Hierarchien.
6. **Desktop Case-Schmiede Export**: Automatische Erstellung von 5 fertigen Dokumenten im Ordner `Desktop/Case_Studies`:
   - `01_Case_Fliesstext.md`
   - `02_Ausfuehrlicher_Report.md`
   - `03_LinkedIn_Posts_10x.md` (inkl. High-Converting Headlines & Chart-Vorschlägen)
   - `04_Meta_Google_Ads_Texts.md` (10x Meta Ads & 10x Google Ads Copywriting)
   - `05_Landingpage_Blogartikel.md` (Step-by-Step Nachbau-Leitfaden)

---

## 🛠 Befehle & CLI-Dashboard

Rufen Sie das Startfenster direkt in der CLI auf mit:
```bash
ai-case-pipeline dashboard
# oder
ai-case-pipeline help
```

Verfügbare Unterbefehle:
- `ai-case-pipeline run` (oder `analyze`): Startet die komplette Case-Schmiede Pipeline.
- `ai-case-pipeline setup`: Konfiguriert die Google Ads API & KI-Einstellungen.
- `ai-case-pipeline scan-periods`: Sucht nach herausragenden Zeiträumen.
- `ai-case-pipeline deep-dive`: Führt die Deep-Dive Feedback-Schleifen durch.
- `ai-case-pipeline status`: Zeigt den aktuellen Konfigurationsstatus.
